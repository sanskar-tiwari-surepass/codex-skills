# uncodex-skill

A frontend design skill for coding agents that enforces non-generic, high-variation UI output.

## Files
- `SKILL.md`: skill behavior, hard constraints, and quality checklist
- `templates/system-prompt.md`: reusable system prompt template for agents

## Focus
- Section variation over repeated templates
- Minimal card usage, no nested cards
- Reference-driven quality from Land-book / Curated patterns
- Strong reuse of Componentry-style components
